# Traffic Replay Library

Traffic Replay Library is a collection of utilities and a replay format parser
for Microserver. It is designed for testing Apache TrafficServer.

## Python Support

This project targets Python 3.10 and newer.

## Development Setup (uv)

1. Install uv: https://docs.astral.sh/uv/
2. Create and sync the environment:

```bash
uv sync --extra dev
```
